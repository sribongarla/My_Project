# Grafana Dashboard Ideas

Create dashboards for:

- Cluster CPU/memory utilization
- Node capacity
- Pod restarts
- Pod evictions
- Deployment replica availability
- Container CPU/memory
- HTTP latency
- HTTP 4xx/5xx error rate
- JVM heap usage
- API throughput



Grafana is the visualization layer. Prometheus collects metrics from Kubernetes and applications, and Grafana displays those metrics in dashboards.
